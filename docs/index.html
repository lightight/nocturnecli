MIT · single binary · macOS · Linux · Windows · amd64 + arm64

Not a chatbot. A working toolchain.

Nocturne drives a real toolset against your project and feeds the actual results back into the conversation. No hallucinated output, no copy-paste relay.

Reads & edits your files
Opens files, makes precise targeted edits, and writes new ones, with every change shown as a diff-style confirmation before anything touches disk.
Runs commands
Executes in your working directory and feeds the real output back.
Searches the codebase
Regex search and directory listing, so it learns structure first.
Takes pasted images
Ctrl+V a screenshot, or reference an image path inline.
One static binary
macOS, Linux, and Windows, on amd64 and arm64. No runtime to install.

One command away

Pick your platform. The installer fetches a prebuilt binary and falls back to building from source if you have Go.

$ curl -fsSL https://nocturnecli.lol/install.sh | sh
PS> irm https://nocturnecli.lol/install.ps1 | iex
$ git clone https://github.com/lightight/nocturnecli && cd nocturnecli $ make install # → ~/.local/bin/nocturne

Then launch it. No environment variables to set:

$ nocturne

Inside the app, run /key noct_your_key once. Your key is saved privately on your machine (~/.config/nocturne) and reused for every project, with no export and no per-folder .env.

Default model is navy:gpt-5.5. Switch any time with /model <id>; model access is per-account.

Slash commands

Everything you need without leaving the prompt.

/helpList every command.
/key [noct_…]Save your API key privately, reused across every project.
/model [id]Show or change the model.
/levelThinking effort: off, normal, or extended.
/streamToggle live token streaming (on by default).
/image <path>Attach an image (a file path, or Ctrl+V from the clipboard).
/autoToggle auto-accept for edits & commands.
/cd <dir>Change the working directory.
/tokensShow token usage & daily quota.
/resumeResume a saved chat from this directory.
/remoteDrive the session from your browser, end-to-end encrypted.
/initGenerate a NOCTURNE.md for the project.
/updateUpdate Nocturne to the latest release.
/clearClear the conversation.

A clean agent loop

The model asks for a tool, Nocturne runs it for real, and the result flows back, looping until the task is done.

  1. You asknatural language
  2. Model calls a toolread · edit · run
  3. Nocturne executeson your machine
  4. Result loops backuntil finished
